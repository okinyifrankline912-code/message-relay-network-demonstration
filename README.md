# Message Relay Network Simulator

## 📡 Overview
This project simulates how data is transmitted across a decentralized communication network using packet switching principles. Messages are broken into packets, routed through multiple nodes, and then reconstructed at the destination.

---

## 🧠 What this project demonstrates
- Computer networking fundamentals (packet switching)
- Randomized routing through nodes
- Packet loss simulation (real-world network behavior)
- Message reconstruction from partial data
- Basic data visualization of network behavior

---

## ⚙️ Features
- Message splitting into packets
- Random node-based routing system
- Packet loss simulation (10% probability)
- Variable network delay
- Message reconstruction
- Histogram visualization of network hops

---

## 🧪 Technologies Used
- Python 3
- Matplotlib
- Random module
- Time module

---

## 📊 Visualization
The program generates a histogram showing the distribution of packet hop counts across the simulated network.

---

## 🚀 Purpose
This project was built to explore how modern communication systems transmit data efficiently across networks and to simulate real-world networking behavior using Python.
